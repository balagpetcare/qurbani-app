## Important Usage Rule

Generated screenshots-গুলো real form হিসেবে বসাবেন না। Request form, doctor list, area select — এগুলো actual Flutter/Android widgets দিয়ে বানাতে হবে। ছবিগুলো style reference এবং Play Store screenshot হিসেবে ব্যবহার করবেন।

## Privacy Rule

Customer-facing public page বা screen-এ doctor phone/WhatsApp number দেখানো যাবে না। Lead access/assignment flow-এর পরেই contact details দেখা যাবে।
